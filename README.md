# Hate Speech Detection using Deep Neural Network

## Overview
This project detects hate speech in social media text using NLP and Deep Learning.

## Technologies
- Python
- TensorFlow
- Bidirectional LSTM
- NLP
- Gradio

## Model Architecture
- Text Vectorization
- Embedding Layer
- Bidirectional LSTM
- Dense Layers
- Sigmoid Output

## Performance
- Accuracy: 0.83
- Precision: 0.88
- Recall: 0.92

## Dataset
Kaggle Toxic Comment Classification Dataset

## How to Run Locally

1. Install dependencies  
   pip install -r requirements.txt  

2. Run the application  
   python app.py  

3. Open the Gradio interface in browser
